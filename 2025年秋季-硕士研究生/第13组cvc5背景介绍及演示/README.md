## 工作简介
cvc5 是一个开源的自动定理证明器，用于解决“可满足性模理论（SMT）”问题，并支持大量的理论及其组合。用于逻辑推理、自动化问题解决等。我们完成了以下内容：

## 思考
通过探索cvc5的python接口以及在多种任务上的使用，发现了cvc5其实在许多实际场景中有潜在用途。比如程序验证，确保一段代码的所有路径满足某些逻辑条件。以及约束优化问题，配合理论求解器，可以处理更复杂的混合逻辑。

但也发现了一些cvc5的缺点，比如建模难度高，必须将问题精确地形式化表达。同时在求解复杂问题时，有可能复杂度很高。所以面对不同的问题选择合适的方式即可。
