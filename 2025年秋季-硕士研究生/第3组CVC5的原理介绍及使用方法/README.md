## 简介
CVC5 是一个现代化的、开源的自动定理证明器，属于 SMT（Satisfiability Modulo Theories，可满足性模理论） 求解器的范畴。

## 工作内容
本项目旨在深入理解 cvc5 的运行机制并探索其应用场景，具体工作如下：
- 理论研究：深入调研了 cvc5 的发展背景及其底层的 SMT 求解原理。
- 实践探索：结合课堂所学知识，编写了多个 Python Demo，验证了 cvc5 在不同类型问题上的求解能力。
- 分析思考：针对求解过程中的现象（如效率、结果形式）进行了深入的分析与反思。

## 讲解视频

视频中我使用几个示例展示了如何使用 cvc5 提供的 python API 解决程序合成（SyGuS），不等式多解问题，图着色多解问题，多元线程方程问题 ，范式求解 ，补码溢出检测， 银行家算法问题。
