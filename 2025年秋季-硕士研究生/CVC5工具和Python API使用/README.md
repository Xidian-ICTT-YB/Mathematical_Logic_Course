## 思考
CVC5就像给代码做数学证明的工具。传统测试是"试几个例子看对不对"，CVC5是"用逻辑推理证明在所有情况下都对"。cvc5很强大，但需要先学会把代码逻辑翻译成数学公式，经过本次学习，掌握了使用cvc5求解器解决问题，并对SAT、SMT的相关理论进行了深入了解。
