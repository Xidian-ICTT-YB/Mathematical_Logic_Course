# cvc5 的介绍与使用-小组1

## 简介

cvc5 是一款高效的开源自动定理证明器，专用于可满足性模理论（SMT）问题。它能够验证一阶公式在多种实用背景理论（及其组合）下的可满足性（或对偶的有效性）。此外，cvc5 还提供语法引导合成（SyGuS）引擎，用于基于背景理论及其组合进行函数合成。

## 示例代码

相关代码放置在 `demo` 目录下，并使用 [uv](https://docs.astral.sh/uv/) 进行管理。在运行所有示例前，需要安装 uv 并在 `demo` 目录执行如下命令：

```sh
uv sync
```

然后，便能够使用 `uv run xxx.py` 命令运行相关示例。代码简介如下：

|      文件       |        描述        |
| :-------------: | :----------------: |
| `arithmetic.py` |  求解基本算数问题  |
|   `logic.py`    |  求解基本逻辑问题  |
|  `overflow.py`  | 求解位向量溢出问题 |
|   `verify.py`   |    执行定理证明    |
|   `zebra.py`    |    求解斑马谜题    |
