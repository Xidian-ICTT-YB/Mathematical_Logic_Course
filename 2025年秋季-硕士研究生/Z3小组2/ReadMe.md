## 思考
如上所说，我们在最后讨论了Z3与目前最火热的大语言模型研究方向的扬长避短结合的可能，大语言模型尽管学习到了丰富的语言表述能力，但相对于理想中的类脑智能来说，缺少必要的数理逻辑能力，如何将Z3求解插入到大语言模型之中，以实现良好的逻辑能力对于目前的类脑智能实现当然是大有意义的。
除此之外，Z3本身的存在对于CS和CV是有一定启示意义的，目前的各个方向的研究都着重于某些方法的无脑堆叠，缺少方法的优美性，或许真正的零到一的突破还是需要我们回到最初的数理逻辑上进行变革
