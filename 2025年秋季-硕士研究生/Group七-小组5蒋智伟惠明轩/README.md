# 基于TinyTroupe的多智能体行为仿真研究 —— 第七小组大作业

**课程**：计算机科学与数理逻辑（2025年秋季 - 硕士研究生）  
**小组成员**：蒋智伟25030020037G 惠明轩25031111082 
**完成日期**：2025年12月

## 🎯 项目简介
本项目基于微软研究院的 **TinyTroupe** 工具包，完成了一次完整的、面向人类行为仿真的多智能体模拟实践。我们模拟了CEO Oscar与设计师Lisa关于"智能冥想舱"产品概念的专业研讨会，展示了LLM在人格模拟、程序化控制及自动化实验方面的强大能力。
