# LLM + cvc5 快速上手指南

## 5分钟快速体验

### 1. 安装依赖(如果还没装)

```bash
pip install cvc5 openai
```

### 2. 直接运行(API已预配置)

```bash
cd examples/llm_integration

# 命令行模式 - 测试单个问题
python nl2smt_solver.py "找三个正整数,和为15,第一个数是偶数"

# 交互模式 - 连续提问
python nl2smt_solver.py
```

### 3. 运行演示脚本(推荐用于PPT/视频演示)

```bash
./demo.sh
```

演示脚本会依次展示4个示例:
1. 整数约束问题
2. 简单方程
3. 数学问题
4. 逻辑推理

## 常见问题

### Q1: 提示"API调用失败"
**A**: 检查网络连接,确保可以访问 https://api.csun.site/

### Q2: 生成的代码执行出错
**A**: 这是正常的,LLM偶尔会生成错误代码。可以:
1. 换个说法重新提问
2. 简化问题描述
3. 检查生成的代码,手动修正

### Q3: 如何更换模型?
**A**: 设置环境变量:
```bash
export OPENAI_MODEL='gpt-4o-mini'  # 或其他支持的模型
python nl2smt_solver.py
```

### Q4: 求解很慢或超时
**A**: 有些问题(特别是需要枚举多个解的)会比较慢。建议:
1. 限制取值范围
2. 只要求一个解
3. 简化约束条件
