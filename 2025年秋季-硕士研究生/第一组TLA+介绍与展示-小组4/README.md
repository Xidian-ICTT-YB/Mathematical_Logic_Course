## 2. 工具简介

TLA+ 由 Leslie Lamport 创立，是一种专注于系统规范与验证的形式化建模工具。它通过集合论和时序逻辑，将复杂系统的行为用简洁的数学表达式描述出来。配合 TLC 检查器，TLA+ 能自动分析模型的所有可能演化路径，帮助开发者在实现前发现潜在的设计问题，显著提升系统的正确性和鲁棒性。
