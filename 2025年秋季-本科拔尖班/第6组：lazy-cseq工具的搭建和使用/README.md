### 操作步骤

1. 在 [Release CSeq 3.0 (SV-COMP 2022) · omainv/cseq](https://github.com/omainv/cseq/releases/tag/SVCOMP2022) 下载最新版本的 CSeq 3.0

2. 解压后即可使用。使用命令：

   ```
   LC_ALL=C ./cseq.py -l lazy -i <target>
   ```
