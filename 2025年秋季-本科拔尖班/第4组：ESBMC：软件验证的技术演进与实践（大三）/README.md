## 思考
ESBMC 代表了软件质量保障技术发展的一个重要方向——从依赖执行的测试，走向基于逻辑的证明。它将复杂的程序逻辑转化为形式化的数学问题，并利用 SMT 求解器提供的自动化推理能力来找到 Bug（SAT）或提供安全保证（UNSAT）。通过掌握 GOTO IR、符号执行、路径条件以及 SMT 编码这些核心概念，我们能更好地理解如何将形式化验证工具应用于工业实践，从而为构建高可靠性、高安全性的软件系统提供坚实的基础。
